# ai-skills

A community-driven collection of AI coding agent skills. Works with any agent supported by the [skills](https://github.com/vercel-labs/skills) ecosystem (40+ agents), and as a native Claude Code plugin marketplace.

## Install

### Any agent (via npx skills)

```bash
npx skills add mmcardle/ai-skills
```

### Claude Code (via plugin marketplace)

```bash
claude plugin marketplace add mmcardle/ai-skills
claude plugin install review-pr-changes@mmcardle-ai-skills
```

## Available Skills

| Skill | Description |
|-------|-------------|
| [review-pr-changes](skills/review-pr-changes) | Review PR changes with structured checklist for quality and standards compliance |
