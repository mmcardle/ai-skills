# Example Skill

This is a template skill demonstrating the standard format for the ai-skills repository. Copy this directory as a starting point when creating new skills.

## When to Use

Describe the trigger conditions here. Be specific about when an agent should activate this skill. For example:

- When the user asks to "do X"
- When the task involves Y
- When working with Z files or patterns

## When Not to Use

- When the task is unrelated to this skill's purpose
- When another more specific skill covers the use case

## Instructions

1. First, assess the situation
2. Then, take the appropriate action
3. Finally, verify the result

## Key Principles

- Keep skills focused on a single purpose
- Write agent-agnostic instructions
- Be specific about activation triggers
- Include both "when to use" and "when not to use" sections
